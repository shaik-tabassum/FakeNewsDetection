Fake News Detection System (Unsupervised Machine Learning)
Project Overview
This project focuses on detecting fake news using unsupervised machine learning techniques. The system analyzes news articles and identifies patterns in the text to classify whether the news content is likely to be fake or real without relying on labeled training data.
Features
Detects fake news from textual news data
Uses Natural Language Processing (NLP) techniques
Implements unsupervised machine learning algorithms
Text preprocessing and feature extraction
Model evaluation and visualization
